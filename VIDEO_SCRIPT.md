## Notas de Producción

| Aspecto | Recomendación |
|---|---|
| **Música** | Track corporativo moderno, ritmo medio-alto, energético pero no intrusivo. Estilo: electro-acoustic corporate. |
| **Fuentes** | Utilizar la fuente del sistema (Geist/Inter) para mantener coherencia de marca. |
| **Colores** | Naranja (#ff6b00) para acentos y CTAs. Negro para fondos. Blanco para texto. |
| **Duración ideal** | 2:00 - 2:30. No exceder 3 minutos. |
| **Formato de salida** | 1920×1080, 30fps, MP4 H.264. |
| **Herramientas sugeridas** | Screen Studio (grabación), CapCut o DaVinci Resolve (edición), Canva (textos y overlays). |

# Ejercicios — Power Automate & RPA

## Ejercicio 1: Flujo de aprobación básico
Crea un flujo que:
1. Se dispare cuando llegue un email a una carpeta específica
2. Envíe una aprobación a un supervisor
3. Si se aprueba, guarde el archivo en SharePoint
4. Envíe una notificación al solicitante

**Herramientas**: Power Automate cloud

## Ejercicio 2: RPA con agent-browser
Usa `agent-browser` para automatizar una tarea repetitiva en un portal web:
```bash
agent-browser "ve al portal de facturación, descarga las facturas del mes y guárdalas en una carpeta"
```

## Ejercicio 3: Integración multi-sistema
Diseña un flujo que:
1. Tome datos de un formulario de Google Forms
2. Los registre en una hoja de Excel en OneDrive
3. Cree un registro en una base de datos SQL
4. Envíe un resumen por Teams

## Ejercicio 4: Automatización con eyes-cli
Pídele a eyes-cli que te explique un concepto:
```bash
eyes "Explica cómo funciona Power Automate Desktop en 3 párrafos, como si fuera para un gerente no técnico"
```

## Checklist de aprendizaje

- [ ] Puedo crear un flujo de aprobación simple
- [ ] Sé qué es RPA y para qué sirve
- [ ] Puedo identificar procesos automatizables en una empresa

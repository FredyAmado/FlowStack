# Módulo 3: Inteligencia Artificial

## Qué aprenderás
- Fundamentos de IA generativa
- Uso de APIs de IA (OpenRouter)
- Visión computacional con modelos multimodales
- Creación de chatbots y asistentes virtuales
- Análisis predictivo básico

## Conceptos clave

### Modelos de lenguaje (LLMs)
- **DeepSeek V4 Flash**: modelo de texto rápido y eficiente
- **Nemotron 3 Nano Omni**: modelo multimodal (texto + visión + audio)
- **Gemini 3.1 Flash Lite**: visión de alta calidad (fallback pago)

### Casos de uso empresarial
- Chatbots de atención al cliente
- Análisis de documentos y contratos
- Extracción de datos de facturas
- Generación de reportes automáticos
- Clasificación de tickets de soporte

## Herramientas instaladas
```bash
# Chat con IA
eyes "¿Qué es la automatización inteligente?"

# Análisis de imágenes
```

# Ejercicios — Inteligencia Artificial

## Ejercicio 1: Chat básico con IA
Usa eyes-cli para diferentes tareas:
```bash
eyes "Escribe un email profesional para un cliente explicando los beneficios de la automatización"
eyes "Traduce al inglés: 'Necesitamos aprobar este presupuesto antes del viernes'"
eyes "Resume este texto en 3 puntos clave: [pega un texto largo]"
```

## Ejercicio 2: Análisis de imágenes
Toma capturas de pantalla de dashboards, facturas o documentos y analízalas:
```bash
eyes captura-dashboard.png "¿Qué KPIs ves en esta imagen? Dame un resumen"
eyes factura-ejemplo.png "Extrae: proveedor, fecha, total, y número de factura"
```

## Ejercicio 3: Generación de imágenes
Crea imágenes para proyectos reales:
```bash
design "banner corporativo moderno con formas geométricas azules y naranjas"
design "logo minimalista para empresa de automatización, estilo tecnológico"
```

## Ejercicio 4: Creación de contenido
Usa campaign-cli para marketing real:
```bash
campaign --brief "Servicios de automatización para empresas logísticas en Colombia"
campaign --post "Lanzamos nuestra nueva plataforma de automatización de procesos"
```

## Ejercicio 5: Sistema de recomendación simple
Pídele a eyes-cli que analice un problema:
```bash
eyes "Tengo una empresa que recibe 200 solicitudes de facturación al día. Sugiere 3 formas de automatizar este proceso con herramientas que existen hoy"
```

# Ejercicios — Automatización Integral

## Ejercicio 1: Arquitectura de solución
Diseña en papel (o con eyes-cli) la arquitectura para automatizar el proceso de:
"Una empresa recibe solicitudes de vacaciones por email, el supervisor las aprueba manualmente, y RH las registra en Excel"

Usa:
```bash
eyes "Diseña una solución de automatización para solicitudes de vacaciones que incluya Power Automate, Power BI e IA. Describe cada paso y qué herramienta usar"
```

## Ejercicio 2: Pitch de venta
Genera un discurso de venta:
```bash
campaign --brief "Vender automatización de facturación a gerente financiero de empresa mediana en Bogotá"
```

## Ejercicio 3: Aprobaciones multi-paso (FlowStack)
Haz una prueba real en FlowStack:
1. Inicia sesión en el dashboard
2. Crea un proceso de aprobación
3. Envía una solicitud
4. Revísala en la bandeja de tareas
5. Revisa el historial

## Ejercicio 4: Dashboard de resultados
Crea un dashboard en Power BI que muestre:
- Procesos automatizados vs manuales
- Tiempo ahorrado por proceso
- Costo evitado por automatización
- Satisfacción del usuario

## Ejercicio 5: Proyecto final
Combina todo lo aprendido:
1. Identifica un proceso real que pueda automatizarse
2. Diseña la solución (diagrama + tecnologías)
3. Implementa un prototipo con agent-browser + eyes-cli
4. Crea un dashboard en Power BI
5. Genera un pitch de venta con campaign-cli
6. Presenta el proyecto como si fuera para un cliente

## Checklist de aprendizaje

- [ ] Sé calcular y comunicar el ROI de un proyecto
- [ ] Puedo presentar propuestas a clientes
- [ ] Integro Power Automate + Power BI + IA en un solo flujo
- [ ] Estoy listo para vender y entregar proyectos reales

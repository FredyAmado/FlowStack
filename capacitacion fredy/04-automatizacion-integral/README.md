# Módulo 4: Automatización Integral

## Qué aprenderás
- Combinar Power Automate, Power BI e IA en soluciones reales
- Arquitectura de soluciones de automatización
- Casos prácticos por industria
- Presentación y venta de proyectos de automatización

## Casos prácticos

### Caso 1: Automatización de facturación
```
Entrada: Email con factura PDF adjunta
Power Automate detecta → Extrae datos con IA → Valida contra ERP
→ Si OK: guarda en SharePoint + registra en contabilidad
→ Notifica por Teams al contador
Power BI: Dashboard de facturas pendientes, pagadas, vencidas
```

### Caso 2: CRM automatizado
```
Formulario web → Power Automate crea lead en CRM
→ Envía email de bienvenido → Agenda tarea en Planner
→ IA clasifica lead (caliente/frío)
→ Power BI: pipeline de ventas en tiempo real
```

### Caso 3: Reporte ejecutivo automático
```
Power Automate (programado cada lunes):
  → Extrae datos de SQL + CRM + Excel
  → Power BI actualiza dashboard
  → IA genera resumen ejecutivo en texto
  → Envía PDF por email a gerentes
```

## Metodología para vender automatización

1. **Descubrimiento**: identifica procesos repetitivos que consumen tiempo
2. **Diagnóstico**: mide cuánto tiempo/dinero se pierde
3. **Propuesta**: diseña la solución con tecnologías concretas
4. **ROI**: calcula el retorno de inversión (ej: "ahorra 20h/semana = $X/mes")
5. **Implementación**: construye el flujo, dashboard o chatbot
6. **Seguimiento**: mide resultados con Power BI

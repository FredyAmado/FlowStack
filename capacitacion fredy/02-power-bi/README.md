# Módulo 2: Power BI & Business Intelligence

## Qué aprenderás
- Conectar fuentes de datos a Power BI
- Modelado de datos con DAX
- Crear dashboards interactivos
- Publicar y compartir reportes

## Conceptos clave

### Power BI Desktop
- **Power Query**: transformación y limpieza de datos
- **Modelo de datos**: relaciones entre tablas
- **DAX (Data Analysis Expressions)**: fórmulas para cálculos avanzados
- **Visualizaciones**: gráficos, tablas, mapas, KPIs

### Power BI Service
- Publicación de reportes en la nube
- Dashboards compartidos
- Actualizaciones programadas

## Fórmulas DAX esenciales
```
SUM()               → Suma de valores
CALCULATE()         → Filtra contexto
IF()                → Condiciones
SUMX()              → Suma por fila
DIVIDE()            → División segura
TOTALYTD()          → Acumulado año a la fecha
```
